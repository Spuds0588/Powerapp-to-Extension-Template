Browser Data Scanners

Tools to identify DOM IDs and Local Storage keys.

These bookmarklets generate CSV files containing the data structures needed to configure the Powerapp-to-Extension template.

View GitHub Repo

How to install

These tools are Bookmarkletsâ€”tiny programs stored inside a browser bookmark.

  1. Ensure your browser's Bookmarks Bar is visible (Ctrl+Shift+B or Cmd+Shift+B).
  2. Click and drag the large buttons below directly onto your Bookmarks Bar.
  3. Navigate to the external website you want to integrate with.
  4. Click the bookmark in your bar to run the scan and download the CSV.

ID Scanner

Map UI Elements

Scans the current page for all HTML elements containing an id attribute.

Use case: Find input field IDs to pass as parameters into your PowerApp via the sidebar iframe.

ID Scanner

Drag this button to your bookmarks bar

Storage Scanner

Map Data Keys

Scans the browser's localStorage for all keys, values, and data types.

Use case: Identify session tokens, user IDs, or settings to reference via local storage keys in your extension.

Storage Scanner

Drag this button to your bookmarks bar
